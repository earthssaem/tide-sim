# 🌊 위도별 조석 형태 시뮬레이터

> **고등학교 지구과학Ⅱ 수업**을 위해 제작된 교육용 시뮬레이터입니다.

**평형 조석론(Equilibrium Tide Theory)** 을 기반으로, 관측자의 **위도**와 달의 **적위**에 따라
하루 동안 해수면 높이가 어떻게 변하는지(조석 형태)를 직관적으로 보여주는 웹 기반 교육용 시뮬레이터입니다.

지구·달·조석 융기를 **3D 모형**으로, 24시간 해수면 변화를 **그래프**로 동시에 확인할 수 있습니다.

---

## ✨ 주요 기능

- **관측자 위도 조절 (0°~90°)** — 저위도·중위도·고위도 구간을 색으로 구분하여 표시
- **달의 적위 조절 (0°~28.5°)** — 조석 융기 중심축의 기울기가 실시간으로 변화
- **관측 시각 조절 (0~24h)** — 슬라이더 직접 조작 또는 재생 버튼으로 자동 진행
- **조석 형태 자동 판별** — 반일주조 · 혼합조 · 일주조 분류
- **3D 시각화** — 지구 자전, 조석 융기, 달 방향, 관측자 위치를 입체적으로 표현 (위아래 드래그로 시점 변경)
- **만조 / 간조 순간 표시** — 3D 모형과 그래프에 동기화된 라벨이 반짝 표시
- **반응형 디자인** — 데스크톱 · 태블릿 · 모바일 화면에 모두 대응

## 📐 이론적 배경

평형 조석론에 따라 해수면 높이 변화 `H(t)` 를 두 성분의 합으로 단순화하여 계산합니다.

```
H(t) = As · cos(4πt / 24)  +  Ad · cos(2πt / 24)
```

| 성분 | 의미 | 진폭 |
|------|------|------|
| `As` | 반일주조 성분 (하루 2회) | `As = cos²φ · cos²δ` |
| `Ad` | 일주조 성분 (하루 1회) | `Ad = sin2φ · sin2δ` |

- `φ` : 관측자 위도, `δ` : 달의 적위
- **조석 형태 판별 지수** `F = |Ad| / |As|`
  - `F < 0.25` → **반일주조**
  - `0.25 ≤ F < 3.0` → **혼합조**
  - `F ≥ 3.0` → **일주조**

> ※ 실제 조석은 달의 공전으로 매일 약 50분씩 늦어지지만, 본 시뮬레이터는 위도별 기본 형태를
> 직관적으로 보여주기 위해 24시간 기준으로 단순화하였습니다.
> 또한 3D 모형의 조석 융기 크기는 이해를 돕기 위해 실제보다 과장되어 있습니다.

## 🚀 사용 방법

별도의 설치나 빌드 과정이 필요 없습니다.

1. `index.html` 파일을 웹 브라우저(Chrome, Edge 등 최신 브라우저 권장)에서 엽니다.
2. 위도 · 적위 · 시각 슬라이더를 조절하며 조석 형태의 변화를 관찰합니다.
3. **▶ 재생** 버튼으로 시간을 자동 진행하거나, **↺ 초기화** 버튼으로 처음 상태로 되돌립니다.

> 인터넷 연결이 필요합니다 (글꼴 및 라이브러리를 CDN에서 불러옵니다).
