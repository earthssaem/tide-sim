# 🌊 위도별 조석 형태 시뮬레이터

관측자의 **위도**와 **달의 적위**에 따라 하루 동안 해수면이 어떻게 변하는지를 3D 모형과 그래프로 동시에 탐구할 수 있는 교육용 웹 시뮬레이터입니다. 
평형 조석론(Equilibrium Tidal Theory)에 기반하며, 고등학교 지구과학2 수업을 위해 제작하였습니다.

## ✨ 주요 기능

### 3D 지구 모형
- 지구 · 달 · 조석 융기(해수층)를 입체적으로 표현
- 조석 융기는 **달 방향과 그 반대 방향, 두 곳**에 형성 → "하루에 만조가 두 번 일어나는" 원리를 직관적으로 표현
- 관측자(빨간 점)가 지구 자전을 따라 위도선 위를 이동
- 위아래 드래그로 **적도 시점 ↔ 극(북극) 시점** 자유롭게 전환
- 관측자가 만조 · 간조 순간을 지날 때 머리 위로 **▲ 만조 / ▼ 간조** 표시가 반짝

### 해수면 높이 변화 그래프
- 24시간 동안의 해수면 높이 변화를 실시간 곡선으로 표시
- 현재 시각의 해수면 위치에 **빨간 원 마커** 표시
- **평균 해수면(0)** 기준선을 점선으로 표시 → 평균보다 높은지 낮은지 직관적 파악
- 3D 모형과 **동기화**되어 만조 · 간조 순간에 그래프에도 동시에 라벨 표시
- 조석 형태(반일주조 / 혼합조 / 일주조)와 만조 최댓값 · 간조 최솟값 자동 표시

### 관측자 시점 표시
- 현재 해수면 변화량 H(t) 실시간 수치 표시
- 위도 구간(저위도 / 중위도 / 고위도) 자동 안내

### 조작 인터페이스
- ▶ 재생 / ↺ 초기화
- 관측자 위도 슬라이더 (0°~90°)
- 달의 적위 슬라이더 (0°~28.5°)
- 관측 시각 슬라이더 (0~24시간, 직접 조작 또는 자동 재생)
- 3D 모형을 마우스/터치로 드래그하여 시점 변경

## 🧭 사용법

1. 좌측 상단 **관측 조건 설정**에서 위도 · 적위 · 시각 슬라이더 조절
2. **▶ 재생** 으로 지구가 자전하며 조석 변화가 진행되는 모습 관찰
3. 3D 모형을 마우스/손가락으로 **위아래 드래그**하여 시점 변경
4. 우측 그래프에서 현재 해수면 위치(빨간 마커)와 24시간 변화 곡선 확인
5. 만조 · 간조 순간 3D 모형과 그래프에 동시에 나타나는 표시로 "지구에서의 순간 = 그래프의 봉우리/골짜기" 연결

## 🎓 교육적 정확성

- 위도(φ)와 달의 적위(δ)로부터 조석 진폭을 계산
  - 반일주조 성분 `As = cos²φ · cos²δ`
  - 일주조 성분 `Ad = sin2φ · sin2δ`
- 해수면 높이 `H(t) = As·cos(4πt/24) + Ad·cos(2πt/24)`
- 두 성분의 비율(Form Factor) `F = |Ad| / |As|` 로 조석 형태 분류
  - `F < 0.25` → **반일주조**
  - `0.25 ≤ F < 3.0` → **혼합조**
  - `F ≥ 3.0` → **일주조**
- 조석 융기는 달 방향 단위벡터와 표면 법선의 내적으로 르장드르 다항식 `P₂(cosθ) = (3cos²θ−1)/2` 을 적용 → 달 방향·반대 방향 양쪽에 융기 형성

> 조석 융기의 크기는 실제 비율보다 크게 과장한 시각화 표현입니다. 또한 실제 조석은 달의 공전으로 매일 약 50분씩 늦어지지만, 본 시뮬레이터는 위도별 기본 형태를 직관적으로 보여주기 위해 24시간을 기준으로 단순화하였습니다.

## 📐 조석 형태와 위도

| 위도 구간 | 일반적 조석 형태 | 특징 |
|---|---|---|
| 저위도 (0°~30°) | 반일주조 경향 | 하루 만조·간조 각 2회, 높이 비슷 |
| 중위도 (30°~60°) | 혼합조 경향 | 두 만조(간조)의 높이 차이 발생 |
| 고위도 (60°~90°) | 일주조 경향 | 하루 만조·간조 각 1회 |

> 실제 조석 형태는 해안 지형·수심 등 여러 요인의 영향을 받습니다. 본 시뮬레이터는 평형 조석론에 따른 기본 경향을 보여줍니다.
